## Lyrics

Ram Ram Ram Parama Sumangala Ram Ram Ram
Janani Janaki Jeevana Ram
Jai Jai Ram Sai Ram Jayatu Jayatu Rama
Parama Sumangala Ram Ram Ram

## Meaning

The name of Ram is highly auspicious. Lord Rama is the life of the universal Mother Janaki (Sita). Victory to Lord Ram and Lord Sai, whose names are highly auspicious.
